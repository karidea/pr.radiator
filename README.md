Fast, local, keyboard-first GitHub PR dashboard for organization teams.

![Example Screenshot](./example-screenshot.png)

## Features
* Open PRs, merged PRs, shortlog, and repository views
* Mnemonic keyboard navigation and filters
* GitHub team-based repository discovery
* Browser-only setup with local configuration. GitHub PAT (`PR_RADIATOR_TOKEN`) required: `repo`, `read:org` scopes (+ SSO for orgs; press `c` to configure).

## Keyboard Shortcuts

### Views
* **l** - repos
* **m** - merged
* **o** - open
* **s** - shortlog

### Filters
* **d** - dependabot
* **f** - cycle author
* **i** - ignore
* **n** - awaiting review
* **t** - cycle team
* **z** - compact

### Navigation
* **Enter** - open
* **G** - bottom
* **gg** - top
* **j/k** - down/up

### Actions
* **?** - help
* **c** - config
* **Esc** - close config
* **r** - refresh PRs
* **R** - refresh repos
