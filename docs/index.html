Keyboard shortcuts

      • j/k: Navigate down/up
      • gg: Jump to first item
      • G: Jump to last item
      • Enter: Open selected item
      • i: Toggle repo ignore (repo view only)
      • d: Toggle Dependabot PRs
      • a: Toggle Merged PRs
      • m: Toggle Master PRs
      • n: Toggle Needs Review PRs
      • l: Toggle Repo Links
      • r: Refresh PRs
      • \: Refresh Repos
      • ?: Show Keyboard shortcuts
